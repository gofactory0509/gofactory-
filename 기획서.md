## 2. 기획 배경 및 목적

### 문제 인식
- 취업 준비생들이 면접 연습할 기회가 부족함
- 실제 면접관으로부터 피드백을 받기 어려움
- 면접 연습을 위해 스터디 그룹을 구하거나 유료 서비스를 이용해야 하는 불편함

### 해결 방안
- AI를 활용하여 **언제 어디서든** 면접 연습이 가능한 서비스 제공
- 직무별 맞춤 질문 생성 및 즉각적인 피드백 제공
- 면접 기록 축적을 통한 성장 과정 추적

---

## 3. 핵심 기능

### 3.1 직무별 AI 면접 질문 생성
- 반도체, 백엔드, 데이터, 마케팅 등 다양한 직무 지원
- Google Gemini AI를 활용한 실전 면접 질문 자동 생성
- 과거 출제 질문과 중복되지 않는 새로운 질문 제공

### 3.2 AI 기반 답변 평가
- **논리성 점수**: 1~100점 척도로 답변의 논리적 구성 평가
- **키워드 분석**: 답변에 포함된 핵심 키워드 확인
- **개선점 제시**: 부족한 부분에 대한 구체적 피드백
- **모범 답변 제공**: AI가 생성한 이상적인 답변 예시

### 3.3 면접 기록 관리
- SQLite 데이터베이스를 활용한 면접 기록 저장 및 조회
- 날짜, 직무, 질문, 답변, 피드백, 점수 기록

### 3.4 직무별 기출 질문 데이터베이스
- 4개 직무 분야 × 7개 질문 = **28개 사전 등록 기출 질문**
- 난이도별 분류 (상/중/하)

### 3.5 오늘의 명언
- 매일 다른 동기부여 명언 제공
- 재방문 유도 및 학습 동기 부여

### 3.6 면접 통계
- 총 연습 횟수 확인
- 주력 직무 분야 파악
- 직무별 연습 분포 시각화
- 평균 점수 추이 확인

## 5. 데이터베이스 설계 (SQLite)

### 5.1 interviews 테이블 (면접 기록)

| 컬럼명 | 타입 | 설명 |
|--------|------|------|
| id | INTEGER (PK) | 자동 증가 기본키 |
| date | TEXT | 면접 일시 |
| job_field | TEXT | 직무 분야 |
| question | TEXT | 면접 질문 |
| answer | TEXT | 사용자 답변 |
| feedback | TEXT | AI 피드백 |
| score | INTEGER | 논리성 점수 (1~100) |

### 5.2 common_questions 테이블 (사전 등록 기출 질문)

| 컬럼명 | 타입 | 설명 |
|--------|------|------|
| id | INTEGER (PK) | 자동 증가 기본키 |
| job_field | TEXT | 직무 분야 |
| question | TEXT | 기출 질문 |
| difficulty | TEXT | 난이도 (상/중/하) |

### 5.3 question_bank 테이블 (AI 생성 질문 축적)

| 컬럼명 | 타입 | 설명 |
|--------|------|------|
| id | INTEGER (PK) | 자동 증가 기본키 |
| job_field | TEXT | 직무 분야 |
| question | TEXT | 질문 내용 |
| source | TEXT | 출처 (기본값: ai_generated) |
| created_at | TEXT | 생성 일시 |

## 6. 사용자 흐름 (User Flow)

```
접속 → 직무 선택 → 면접 시작 → AI 질문 생성 → 답변 작성 → AI 피드백 → 기록 저장 → 다음 질문/종료
```

### 상세 흐름:
1. 사용자가 서비스에 접속
2. 원하는 직무 분야 선택 (반도체/백엔드/데이터/마케팅)
3. "면접 시작" 버튼 클릭
4. AI가 해당 직무에 맞는 면접 질문 생성
5. 사용자가 답변 작성 후 제출
6. AI가 답변을 분석하여 피드백 제공 (점수, 키워드, 개선점, 모범답변)
7. 면접 기록이 데이터베이스에 자동 저장
8. 다음 질문으로 계속하거나 면접 종료

## 8. 향후 발전 방향

- **사용자 계정 시스템 추가**: 로그인/회원가입으로 개인별 데이터 관리
- **면접 점수 랭킹 시스템**: 다른 사용자와 비교하여 동기부여
- **음성 면접 기능 (STT/TTS)**: 실제 면접처럼 음성으로 답변
- **더 많은 직무 기출 질문 확보**: IT, 금융, 컨설팅 등 직무 확대
- **면접 영상 분석**: 표정, 시선, 자세 분석 기능
